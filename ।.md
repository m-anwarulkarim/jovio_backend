# Dashboard Route

## `GET /api/v1/dashboard/stats`

**কাজ:** logged-in user অনুযায়ী dashboard stats আনা

**Access:**

- `ADMIN`
- `COMPANY_OWNER`
- `EMPLOYEE`

**Request:**
কোনো body লাগবে না
শুধু login session / cookie লাগবে

---

## Example Request

```http
GET /api/v1/dashboard/stats
```
